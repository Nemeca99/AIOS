# To run the Codex Web Calculator:
# 1. Open PowerShell in the e:\Algebra\Calculator directory.
# 2. (Optional) Create a virtual environment:
#    python -m venv venv
#    .\venv\Scripts\Activate
# 3. Install Flask and requests:
#    pip install flask requests
# 4. Start the web server:
#    python codex_web_calculator.py
# 5. Open your browser (on phone or desktop) and go to:
#    http://<your-pc-ip>:5000
#    (Use 127.0.0.1:5000 if on the same machine)
#
# The web UI is mobile-friendly and will use your Codex memory and Ollama LLM.
